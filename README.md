## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo
   ```sh
   gh repo clone JensenYuen/rockpaperscissors
   ```
2. Install Ruby gems 
   ```sh
   cd rockpaperscissors
   bundle install
   ```
3. To run the application
   ```sh
   ruby lib/interface.rb
   ```
